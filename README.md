
# ChipCheck

ChipCheck is a lightweight, browser-based poker buy-in and cashflow tracker designed for friendly home games.  
It helps players accurately track buy-ins, rebuys, and total money at the table without disrupting the game flow.

The project focuses on simplicity, speed, and mobile usability.

---

## Features (v1)

- Add players quickly
- Track multiple buy-ins per game
- Automatic total buy-in calculation
- Simple and clean UI optimized for mobile
- No login or setup required

---

## Tech Stack

- **HTML** – Application structure  
- **CSS** – Styling and mobile-friendly layout  
- **JavaScript** – Game logic and calculations  

No backend or database is used in v1.

---

## How to Use

1. Open the application in a web browser
2. Enter a player name and buy-in amount
3. Click **Add Buy-In**
4. View updated totals instantly

---

## How to Use on Mobile

### Option A: GitHub Pages (Recommended)

1. Open the deployed GitHub Pages link in your mobile browser  
2. Use the app directly during the poker game  
3. No installation required  

### Option B: Local Network (Live Server)

1. Open the project in VS Code  
2. Install the **Live Server** extension  
3. Run Live Server  
4. Ensure your phone and laptop are connected to the same Wi-Fi  
5. Open the local URL on your phone browser
